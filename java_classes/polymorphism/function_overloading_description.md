As The name suggests it means overloading the function basically assigning more than one work to a function.

It is the also known as compile time polymorphism.
In this we create many function with same name and assign them different work.

Things to be keep in mind while implementing overloading : 

- There should be a differentiating factor in this like either the return type of the function should be different from each other if there is nothing different,

- If the retrun type is same than the parameter should be different from each other.

- Or if the both the things are same than there should be difference between the attribute of the function.
